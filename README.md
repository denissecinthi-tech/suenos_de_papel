# Sueños de Papel

Bienvenido al repositorio de **Sueños de Papel**, una página web para nuestra tienda de papelería. 

# Características
* Catálogo dinámico de productos de papelería básica, escolar y técnica.
* Carrito de compras funcional con cálculo de totales.
* Formulario de contacto interactivo con validaciones y opción de envío a WhatsApp.
* Sección de preguntas frecuentes interactiva.

# Tecnologías utilizadas
* HTML5
* CSS3 
* JavaScript 
* Bootstrap 5 & Font Awesome
